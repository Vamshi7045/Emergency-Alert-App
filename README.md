# Emergency-Alert-App
Android emergency alert application that sends SMS with GPS location and automatically calls emergency contact when SOS button is pressed.

## Features

- SOS button for quick emergency alert
- Sends SMS with current GPS location
- Automatically calls the emergency contact
- Simple and user-friendly interface

## Technologies Used

- Kotlin
- Android Studio
- Jetpack Compose
- GPS Location API
- SMS Manager

## How It Works

1. User presses the **SOS button**
2. App fetches the current GPS location
3. SMS with Google Maps link is sent to the emergency contact
4. Phone automatically calls the emergency number
